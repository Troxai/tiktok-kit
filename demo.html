1

OAuth Authentication

Required

User authorizes via TikTok OAuth 2.0. Redirect to TikTok login:

GET https://www.tiktok.com/v2/auth/authorize/ ?client_key=YOUR_CLIENT_KEY &redirect_uri=https://tiktok.tnis.cloud/callback &scope=user.info.basic,video.upload,video.publish,comment.list,video.insights &state=random_state_string

After user approves → TikTok redirects back with auth code → exchange for access token:

POST https://open.tiktokapis.com/v2/oauth/token/ { "client_key": "YOUR_CLIENT_KEY", "client_secret": "YOUR_CLIENT_SECRET", "code": "authorization_code", "grant_type": "authorization_code", "redirect_uri": "https://tiktok.tnis.cloud/callback" }
user.info.basic video.upload video.publish comment.list video.insights

Video Upload & Publish

video.upload / video.publish

User sends video via our backend API to TikTok:

POST Our API /api/v1/video/upload Authorization: Bearer <access_token> { "video_file": "[binary video data]", "caption": "Check out my new video! #fyp", "privacy_level": "PUBLIC" } → Response: { "status": "success", "video_id": "7439281756394872838", "share_url": "https://www.tiktok.com/@user/video/7439281756394872838" }
